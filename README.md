# DouyuBarrage

斗鱼弹幕抓取及实时弹幕数据可视化，分为crawler(弹幕抓取)，server(弹幕统计数据服务器)，web(统计数据可视化前端)三部分。

![预览](https://github.com/Crawler995/DouyuBarrage/blob/master/doc/preview.png)

## 运行

目前处于开发阶段，需手动逐个启动各模块。启动顺序一般为

1. 启动crawler

2. 启动server

3. 启动web

## 相关技术

1. crawler

   Python 3.7

   Mysql

2. server

   Java 8

   Spring Boot

   Mybatis

3. web

   JavaScript

   React

   Ant Design

   Echarts
